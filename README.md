# F1-Sim

This project was built on mac, so you may find that local pathways directories might not work due to how different operating systems work.
If this issue occurs on a windows device, usual fix is to switch from forward slashes (/) to backward slashes (\) in any file pathways.

For this to work you will also need to install the pygame libary, and FastF1 if you wish to store the full race data by running the build_fastF1_cache.py in the tools folder. If so run run this file first before any other files in the tools folder, and run it at least 3 times due to possible website time outs and caching errors.
